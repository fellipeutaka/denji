Denji is a CLI tool that simplifies icon management in React projects. It fetches icons from Iconify's vast library, converts them to optimized React components, and maintains a centralized icons file.

## Features

- **200,000+ icons** - Access icons from Iconify's collection (Lucide, Material Design, Heroicons, etc.)
- **Type-safe** - Full TypeScript support with auto-completion
- **Optimized** - Generates clean, tree-shakeable React components
- **Accessible** - Built-in accessibility strategies (ARIA attributes, titles, roles)
- **Customizable** - Lifecycle hooks for custom transformations
- **Zero dependencies** - Lightweight CLI with no runtime overhead

## About the Name

Named after Denji from Chainsaw Man. Short, memorable, perfect for CLI commands.

![Denji from Chainsaw Man](https://media1.tenor.com/m/9TCYYMfVlYAAAAAd/denji-chainsaw-man.gif)

## Quick Start

<Steps>
<Step>

### Install Denji

```npm
npm i -D denji
```

</Step>

<Step>

### Initialize project

```bash
denji init
```

Creates `denji.json` config and icons template file.

</Step>

<Step>

### Add icons

```bash
denji add lucide:check mdi:home
```

Fetches icons from Iconify and adds to your project.

</Step>

<Step>

### Use in React

```tsx
import { Icons } from "./icons";

function App() {
  return <Icons.Check className="size-4" />;
}
```

</Step>
</Steps>
